Detailed Description

A neuron network is a set of neurons and a set of directional links between them.

Neurons are objects of class Neuron (they are identified by their index in the vector neurons). A link is an ordered pair of indices in neurons with an intensity value, collected in a linkmap. This is a std::map therefore only one connection can exist between two neurons.

To create a network, you need to resize it and optionally set_default_params for each neuron. Then you can either call add_link for each connection or generate a random network with random_connect.

The dynamics of the network proceeds by calling step. The state of the network can be printed to output streams with print_params (to print all parameters of all neurons), print_traj to print the full state of one neuron of each type. The helper function print_head will print a header line with the variable names for the the print_traj lines.

This class provides accessors to the following Neuron properties

  • degree : returns the degree of a neuron (number of incoming connections) and valence (sum of weights of these links)
  • neighbors : returns the indices of neurons with incoming links to a given neuron,
  • potentials : returns the values of membrane potentials for all neurons,
  • recoveries : returns the values of recovery variables for all neurons,
  • neuron : returns a const reference to a given neuron object.

size_t Network::random_connect ( const double &  mean_deg,
const double &  s = _STRENG_ 
)

Creates random links in the network. Each neuron will expect to receive n connections (RandomNumbers::poisson with mean mean_deg) of intensity s (RandomNumbers::uniform_double with mean mean_streng). Sending neurons will be picked at random and since there can be only one connection from each neuron, the expected degree is not always achieved.

Parameters
mean_deg(double): mean value of Poisson distribution.
mean_streng(double): mean value of the uniform distribution (with bounds 0 and 2*mean_streng).
Returns
the number of links created.
