Detailed Description

A neuron type is defined by four parameters a, b, c, d, and the "inhibitory" or "excitatory" quality.

Standard types are provided in NeuronTypes and identified by a 2 or 3-letter string.

The dynamic variables are the membrane potential, the recovery variable and the input.
